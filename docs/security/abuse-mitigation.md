# Abuse mitigation

Abuse is inevitable in any open system. RCLDX focuses on detection, containment and recovery.

## Types of abuse

- Flooding the cluster with fake spots
- Offensive or illegal content in messages
- Technical DoS against brokers or gateways

## Mitigation techniques

- Rate limiting per user / per callsign
- Grandfathering-based trust and demotion
- Global and local blacklists
- Automated filters for forbidden terms

## Incident response

Core Team has documented procedures to:

- Identify malicious actors
- Revoke tokens and sponsorships
- Share intelligence between clubs and Core Team
