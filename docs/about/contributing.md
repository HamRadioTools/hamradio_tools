# Contributing

RCLDX is open by design. Contributions are welcome.

## How to Contribute

- Report issues and ideas.
- Enter the discussion board and give opinion.
- Use the Pull-Requeste mechanism for:
  - Write documentation.
  - Implement tools and reference clients.
- Operate club MQTT brokers.

## Code of Conduct

Contributors are expected to behave respectfully and constructively.

## Contribution Process

Typical workflow:

```markdown
discuss → design doc → implementation → review → deployment
```
