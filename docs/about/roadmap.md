# Roadmap

This is a living roadmap for RCLDX evolution.

Last update: **2025-12-04**

## Short-term (target Jan 2026)

- Stabilize core protocol and topic structure
- Implement reference filtering engine
- Release basic web dashboard visualization

## Medium-term (target May 2026)

- Integrate the basic web dashboard with CAT
- Add MeshCore / LoRa integrations
- Improve observability and tooling for operators
- Document best practices for clubs

## Long-term (target Sep 2026)

- Wideband IQ-based spotting
- Federation patterns and partial replication
- Open reference implementations in multiple languages
- Write SDK
