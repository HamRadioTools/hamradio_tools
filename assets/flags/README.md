# Flags Assets (MIT)

This folder contains SVG flag assets vendored from:
- Project: `flag-icons`
- Repository: https://github.com/lipis/flag-icons
- License: MIT (see `upstream/flag-icons-LICENSE`)

Source path used:
- `flags/4x3/*.svg`

These assets are stored locally so they can be served directly from GitHub Pages.

## Naming
Files use lowercase codes, for example:
- `us.svg`
- `de.svg`
- `gb-nir.svg`

Code-to-file convention:
- input code (e.g. `GB-NIR`) -> lowercase filename (`gb-nir.svg`)
